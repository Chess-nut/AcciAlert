# AcciAlert
AcciAlert is a mobile app made with React Native. It helps people in the barangay community report road accidents and dangerous spots on the road. When someone reports an incident, the app sends the information to local authorities and emergency teams with the exact location.

Features:
 - Report accidents and road hazards
 - Show location on a map
 - Push notifications for new incidents
 - Offline mode to save reports without internet
 - Verified reports by authorities
 - Comment, share, and upvote incidents

Tech Used:
 - React Native
 - Java Script
